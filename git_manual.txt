Git에 올리는 방법

1. pwd로경로를  확인한다.
2. 경로가 맞지 않다면 cd 기능을 통해 들어가기
3. git init으로 올리는 저장소 지정
4. git config --global user.name ""으로 닉네임 설정
5. git config --global user.email ""로 이메일 설정
6. git config --global user.name으로 닉네임 확인
7. git config --global user.email으로 이메일 확인
8. cat 파일명으로 파일 확인
9. git add 파일명으로 working tree로 보냄
10. git commit -m ""로 staging area로 보냄
11. git push origin 브랜치명으로 git에 업로드
12. git pull origin 브랜치명으로 git에서 데이터 받아오기
13. git clone으로 파일을 그대로 가져오기
